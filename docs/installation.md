## Installation
### Go
```
go install github.com/Karthika-Rajagopal/authz0@latest
```

### Homebrew
```
brew tap Karthika-Rajagopal/authz0
brew install authz0
```

### Snapcraft
```
snap install authz0
```

## Update
### Go
```
go install github.com/Karthika-Rajagopal/authz0@latest
```

### Homebrew
```
brew upgrade authz0
```

### Snapcraft
```
snap refresh authz0
```
